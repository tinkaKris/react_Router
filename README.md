# ДЗ 42. Вступ до маршрутизації в React з використанням React Router

## Мета проекту:

ознайомитися з основами client-side маршрутизації в React за допомогою бібліотеки React Router.

## Задачі проекту:

- встановлення бібліотеки react-router-dom.

- використання компонента BrowserRouter для ініціації маршрутизації у додатку.

- використання компоненти Route для встановлення маршрутів до компонентів і Link для створення навігаційних посилань. Це дозволить організувати базову навігацію між сторінками додатку.

- використання NavLink замість Link для навігаційних елементів в основній панелі навігації. Налаштування NavLink, щоб воно виділяло активне посилання, змінюючи його стиль або клас.

- реалізація вкладеної маршрутизації
